# Monument_DIY

A lightweight editor that gives players full control over monument placement and properties.

**Features:**
- Reposition any monument anywhere on the map
- Edit monument properties (effects, requirements)
- Select monument by images

**Tech Stack:**
- Life is short, use python

**Why I built it:**  
I wanted to explore how game engines handle dynamic object manipulation, and this mod became a hands-on way to learn game file structures and modding workflows.

**usage**
<pre>
step 1 create a virtual environment
    python3 -m venv venv
step 2 activate it
    source venv/bin/activate
step 3 install the necessary extensions
    pip install -r requirements.txt
step 4 run the code
    python front_end.py
</pre>
